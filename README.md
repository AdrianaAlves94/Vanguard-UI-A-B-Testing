# Vanguard A/B Test Analysis

## Project Overview

This project analyzes Vanguard’s redesigned digital client experience using an A/B testing framework.  
As part of Vanguard’s CX Analytics Team, we investigated whether the redesigned interface improved user completion behavior, reduced friction, and created measurable business value.

Clients were split into:
- **Control Group** → existing interface
- **Test Group** → redesigned interface

We evaluated user behavior across the full digital journey using KPIs, exploratory data analysis and hypothesis testing.
---

## Key Questions

- Did the redesigned experience improve completion rates?
- Did users complete the process faster?
- Did the redesign reduce user friction?
- Did the redesign generate measurable business value?

---

## Main Findings

### Improvements
- Completion rate increased from **40.5% → 43.9%**
- Users completed the process **14% faster**
- Stronger progression from **Start → Step 1**
- The redesign exceeded Vanguard’s internal **5% business threshold**

### Challenges
- Error/friction rate increased
- Users experienced more repeated actions and longer journeys
- Step 2 showed signs of usability friction

### Recommendation
We recommend a **phased rollout** with targeted UX improvements and continued monitoring of friction-heavy interactions.
